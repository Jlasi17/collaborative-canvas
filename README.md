# Real-Time Collaborative Drawing Canvas

A multi-user drawing application where multiple people can draw simultaneously on the same canvas with real-time synchronization.

## Features

- **Real-time Drawing**: See other users' drawings as they happen
- **Multiple Tools**: Brush and eraser with adjustable size
- **Color Selection**: Choose from any color for drawing
- **User Indicators**: See other users' cursor positions
- **Global Undo/Redo**: Undo and redo actions across all users
- **User Management**: View who's online with assigned colors
- **Mobile Support**: Touch-enabled for mobile devices

## Setup Instructions

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/Jlasi17/collaborative-canvas.git
cd collaborative-canvas
```
2.Install dependencies
```bash
npm install
```
3.run the server
```bash
npm start
```
After this, the app will be available on your localhost (usually http://localhost:3000).
